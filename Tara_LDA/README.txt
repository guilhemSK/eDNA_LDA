R code associated with the submitted manuscript 'Global drivers of eukaryotic plankton biogeography in the sunlit ocean'.

Multigroup_Tara_analyses.R contains all the scripts used to generate the results presented in the manuscript based on Latent Dirichlet Allocation model outputs. The corresponding results are provided in the Saved_results folder. Tara_figures.R contains all the scripts used to generate the manuscript's main and supplementary figures from the results in Saved_results. Normalized.VI_fun.R is a function called by Multigroup_Tara_analyses.R.

Suitable absolute paths to data, code and figure folders need to be defined at the beginning of Multigroup_Tara_analyses.R and Tara_figures.R.
